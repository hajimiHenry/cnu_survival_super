# 条目: 国家奖学金申请与评审流程
# 类别: 流程
# 主题: 国家奖学金
# 来源: 首都师范大学学生手册整理

第一步 关注每学年初的学校通知  
国家奖学金评审一般在每学年上学期进行 学校学生资助管理部门会通过官网 微信公众号 或院系工作群发布当年评审细则和名额分配  

第二步 学生本人提出申请  
符合条件的学生按通知要求填写奖学金申请表 准备成绩单 综合测评结果 个人事迹材料等 在规定时间内向所在院系递交  

第三步 班级民主评议  
班级或年级资助评议小组讨论申请人的学习表现 思想品德 奉献服务等情况 进行民主评议 只有获得多数同意的申请人才能进入院系推荐名单  

第四步 院系审核与公示  
院系资助工作领导小组对班级推荐学生的材料进行审核 根据学院名额和综合表现提出拟推荐名单 在院系范围内公示 一般不少于三日 公示中如有异议 需重新核查  

第五步 学校评审与公示  
院系将推荐名单和材料报送学生资助管理中心 由学校学生资助工作领导小组进行综合评定 确定最终获奖名单 并在全校范围内公示 一般不少于五日  

第六步 名单报送与资金发放  
公示无异议后 学校将获奖名单报上级部门备案 资金按统一时间发放到学生银行卡 并在学生档案中记录

---

# 条目: 国家助学金及北京市国家助学金申请流程
# 类别: 流程
# 主题: 国家助学金
# 来源: 首都师范大学学生手册整理

第一步 家庭经济困难学生认定  
助学金通常与家庭经济困难认定结果挂钩 学生需在学年初按学校要求提交家庭收入情况 相关证明材料和困难情况说明 经过班级评议 院系审核和学校公示后获得困难等级  

第二步 填写助学金申请  
在学校发布的助学金评定通知规定时间内 符合条件的学生填写国家助学金或北京市国家助学金申请表 说明家庭情况和个人表现  

第三步 班级民主评议  
班级资助评议小组综合考虑学生困难程度 学习态度和日常表现 按照名额提出建议名单  

第四步 院系审核与排序  
院系资助工作领导小组对班级推荐名单进行审核 根据困难等级和综合情况 确定本院系的建议人选和资助档次 在院系范围内公示  

第五步 学校审定与公示  
院系名单报学生资助管理中心 汇总后由学校学生资助工作领导小组审批 并在全校范围内公示 无异议后确定最终资助名单和金额  

第六步 资金发放  
助学金按学年或学期发放到学生个人账户 具体发放节奏由学校财务和资助部门安排

---

# 条目: 社会奖助学金申请流程
# 类别: 流程
# 主题: 社会奖助学金
# 来源: 首都师范大学学生手册整理

第一步 了解各类社会奖助学金条件  
每学年十月中下旬左右 学校会发布当年社会奖助学金评选通知 说明金宝奖学金 尚朴奖学金等各项目的资助对象 特殊要求和名额  

第二步 学生本人提出申请  
符合条件的学生向所在院系提出申请 填写奖助学金申请表 并附上成绩证明 综合测评结果 事迹材料等  

第三步 班级民主评议推荐  
班级资助评议小组组织同学对申请人进行民主评议 只有多数同学同意的申请人 才能作为推荐对象上报院系  

第四步 院系初审与公示  
院系资助工作领导小组审查班级推荐结果 对符合条件的学生在院系范围内公示三日 公示期间如有异议 需重新审查  

第五步 学校审定与公示  
院系将无异议的初审名单 申请表和相关材料报学生资助管理中心 由学校学生资助工作领导小组综合评定 后在全校公示一般五日 公示无异议后向出资单位报送名单  

第六步 资金发放  
社会奖助学金的发放时间和方式会根据资助方要求有所不同 一般在当年十二月底前发放到学生手中

# 条目: 临时困难补助申请流程
# 类别: 流程
# 主题: 临时困难补助
# 来源: 首都师范大学学生手册整理

第一步 明确申请理由  
临时困难补助主要用于应对家庭遭遇突发重大变故 本人重大疾病 自然灾害等造成的短期经济困难 学生需要梳理清楚困难发生的时间 经过和影响  

第二步 提交书面申请材料  
学生向所在院系提交书面申请 说明家庭成员情况 收入来源 突发事件经过 现有支出压力 等 并尽量附上医院诊断 住院票据 灾害证明等材料  

第三步 院系初审  
院系资助工作领导小组对申请材料进行审核 核实困难情况 是否符合学校临时困难补助的范围 并给出建议金额  

第四步 学校审批  
金额在三千元以下的 由院系报学生处审批 金额超过三千元的 由院系报学生处 由学生处和财务处共同核实 后报主管校领导批准  

第五步 发放补助  
审批通过后 学生处填写临时困难补助发放单 由学生本人或院系老师到财务处领取补助款 并在系统中登记相关记录

---

# 条目: 学费与住宿费减免申请流程
# 类别: 流程
# 主题: 学费住宿费减免
# 来源: 首都师范大学学生手册整理

第一步 确认是否属于重点关照对象  
根据减免暂行办法文号学发〔2021〕7号 学费和住宿费减免主要面向家庭经济特别困难 但仍坚持完成学业的全日制在校本专科生 需要具备正式学籍 按时注册 并已缴纳学校各类费用  

第二步 提交减免申请  
学生向所在院系提交学费或住宿费减免申请书 说明家庭经济状况 困难类型 既往获得资助情况 等 并附上困难证明材料  

第三步 院系初审与推荐  
院系资助工作领导小组根据文件条件 对申请人进行审核 并结合学业表现 日常表现和家庭状况提出减免建议比例 和推荐名单  

第四步 学校审定  
院系名单上报学生资助管理中心 由学校组织专项评审 结合年度资助资金总量 统一确定获减免学生名单和具体减免额度  

第五步 执行减免  
减免结果反馈给财务部门 通过冲抵学费或住宿费的方式执行 学生在个人缴费记录中可以看到减免金额

---

# 条目: 校内勤工助学岗位申请流程
# 类别: 流程
# 主题: 勤工助学
# 来源: 首都师范大学学生手册整理

第一步 关注勤工助学岗位发布  
学生资助管理部门或院系会在每学期初通过公告栏 网站或工作群发布校内勤工助学岗位信息 说明岗位类型 工作内容 工作时间和报酬标准  

第二步 提交岗位申请  
有意向的学生结合课程安排 选择适合的岗位 向岗位所在单位提交申请 通常需要填写报名表 并说明家庭经济情况和个人技能  

第三步 岗位单位面试和录用  
用人单位根据岗位需要 对报名学生进行面试或简单交流 确定录用名单 并将结果报学生资助管理部门备案  

第四步 签署勤工助学协议  
录用学生与学校或岗位单位签署勤工助学协议 约定工作时间 工作任务 权利义务和安全要求 确保不影响正常学习  

第五步 上岗与考核  
学生按约定时间到岗 工作期间需要服从管理 认真完成任务 岗位单位按月或按学期进行考核  

第六步 工资发放  
勤工助学报酬按规定标准通过学校财务系统发放 一般打入学生银行卡

---

# 条目: 公费医疗普通门诊报销流程
# 类别: 流程
# 主题: 公费医疗门诊
# 来源: 首都师范大学学生手册整理

第一步 在校医院或指定医院就诊  
享受公费医疗待遇的学生 一般应优先到校医院就诊 遇到急诊或特殊病种时 需按公费医疗管理办法规定的指定医院就医  

第二步 按规定挂号就诊并保存票据  
就诊过程中注意保存挂号单 检查单 药品清单和费用发票 这些都是后续报销所需材料  

第三步 确认是否属于常规项目  
普通门诊一般项目按公费医疗规定比例报销 学生无需额外审批 但自费药物和不在报销范围内的项目费用需要自付  

第四步 办理报销  
在规定时间内 携带学生证 身份证和相关票据 到校医院或学校指定的公费医疗报销窗口办理核销 工作人员按公费医疗管理办法审核后 在系统中进行报销处理  

第五步 如涉及大额检查或贵重材料  
若涉及单项费用在规定额度以上的大型检查或贵重一次性医用材料 需要严格按单独审批流程执行

---

# 条目: 大型检查与贵重医用材料审批流程
# 类别: 流程
# 主题: 公费医疗审批
# 来源: 首都师范大学学生手册整理

第一步 医生提出检查或治疗建议  
当经治医生认为有必要进行大型影像检查 或使用贵重医用材料时 会向学生说明原因 检查和治疗的必要性 以及费用情况  

第二步 填写审批表  
对于单项费用在一定数额以上的大型检查 或贵重一次性医用材料 医生需要填写贵重医用设备及材料检查与治疗审批表  

第三步 本人或家属签字确认  
学生本人或家属在审批表上签字 确认已了解相关费用和治疗方案  

第四步 公费医疗管理部门审批  
审批表报校医院公费医疗管理办公室或业务副院长审批 通过后方可按照公费医疗规定比例报销 否则可能需要自负更高比例的费用  

第五步 检查或治疗与报销  
审批通过后 学生按医生安排进行检查或治疗 结束后凭审批表和费用票据按公费医疗规定办理报销

---

# 条目: 预约心理咨询与危机求助流程
# 类别: 流程
# 主题: 心理咨询
# 来源: 首都师范大学学生手册整理

第一步 判断目前需要的支持类型  
若只是一般情绪困扰 学习压力或人际问题 可以通过预约个体咨询或参加团体辅导获得帮助  
若已经出现明显的自伤念头 行为冲动 或严重功能受损 应视为心理危机 需要尽快联系心理中心或危机热线  

第二步 预约校内心理咨询  
良乡校区的同学可前往艺术楼一层的心理中心值班室 或通过公布的预约电话进行预约 校本部同学可前往大学生活动中心一层西侧心理中心或电话预约  
预约时需要提供姓名 学号 院系 联系方式和大致困扰内容 由中心安排咨询师和时间  

第三步 按时参加咨询  
在约定时间地点接受咨询 按咨询师要求配合填写量表或完成评估 咨询过程一般保密 如涉及自伤他伤风险 咨询师可能需要联系家长或学校相关部门共同干预  

第四步 危机情况下的紧急求助  
若出现强烈自杀冲动或严重危险行为 无法等待预约时间 可在工作时间直接前往心理中心现场求助 或让同学 老师陪同前往  
也可以拨打北京心理危机干预热线或北京青少年服务台 等外部热线 进行二十四小时电话求助  

第五步 医疗转介  
若经心理评估认为需要药物治疗或住院治疗 咨询师可以建议前往合作医院的心理或精神科门诊 就诊时携带学生证和身份证
