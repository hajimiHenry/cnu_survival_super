# 条目: 学生在校权利与义务总览
# 类别: 制度
# 主题: 权利与义务
# 来源: 首都师范大学学生手册整理

在校期间 学生一方面有权利 另一方面也有必须履行的义务  
主要权利包括 参加培养方案里安排的各类教学活动和实践活动 使用学校提供的课堂 图书馆 实验室和网络等教育教学资源 参加社会实践 志愿服务 勤工助学 文娱体育和科技创新活动 并获得就业创业指导  
学生可以申请奖学金 助学金 助学贷款和其他经济资助 在修业进度和选课方面享有一定自主选择权 在思想品德和学业成绩方面应当获得公平公正的评价 完成规定学业后有权取得相应学历证书和学位证书  
在校内可以依法组织和参加学生团体 以适当方式参与学校管理 对涉学生权益事务享有知情权 参与权 表达权和监督权  
对学校作出的处理或处分有异议时 学生有权向校内学生申诉处理委员会和北京市教育委员会提出申诉  
主要义务包括 遵守宪法 法律 法规 遵守学校章程和各种管理规定 诚实守信 恪守学术道德 按时完成培养方案规定的学习任务 按规定缴纳学费和相关费用 履行获得奖助学金和贷款后的相应义务 遵守学生行为规范 尊敬师长 养成良好品行

---

# 条目: 校园秩序与行为底线
# 类别: 制度
# 主题: 校园秩序
# 来源: 首都师范大学学生手册整理

学生和学校共同维护正常的教学 科研和生活秩序 学生应当自觉遵守公民道德规范和学校的各项管理制度 维护文明 整洁 安全的校园环境  
学生不得在校内酗酒 打架斗殴 赌博 吸毒 不得传播 复制 贩卖非法书刊 音像制品和违法有害信息 不得参与非法传销 不得参加邪教和封建迷信活动 不得从事有损大学生形象和违反社会公序良俗的活动  
大型集会 游行 示威等活动必须按法律程序和学校规定事先申请批准 未获批准的 学校可以依法予以劝阻或制止  
学生住宿要遵守公寓管理和消防安全规定 一般不得在宿舍留宿校外人员 不得在宿舍留宿异性 不得违规使用大功率电器 不得私拉乱接电线  
学校坚持教育与宗教相分离原则 任何组织和个人不得在校园内进行宗教活动

---

# 条目: 网络使用与信息发布管理
# 类别: 制度
# 主题: 网络与信息
# 来源: 首都师范大学学生手册整理

学生使用校园网络时 必须遵守国家有关互联网管理规定和学校网络使用制度  
不得登录非法网站 不得在网上传播违法 有害 虚假的文字 图片 音频 视频等信息 不得攻击或非法侵入他人计算机和移动通信网络系统  
在校园内张贴通知 广告等内容 需要在学校指定地点进行 散发宣传品 印刷品应按学校要求报批 对于反对宪法根本制度 破坏国家利益或侮辱诽谤他人的内容 学校和有关部门将依法追究责任  
利用学校广播 电视等公共信息发布设施 需要按程序申请批准 不得擅自占用或干扰正常教学科研秩序

---

# 条目: 学生经济资助体系总览
# 类别: 制度
# 主题: 经济资助总览
# 来源: 首都师范大学学生手册整理

学校建立了比较完整的学生资助制度体系 主要包括以下几类文件  
学生资助管理工作暂行办法 学生综合测评实施办法 家庭经济困难学生认定管理办法 本专科生国家奖学金实施细则 本专科生国家励志奖学金实施细则 学生奖学金实施办法 北京市国家助学金实施细则 社会奖助学金实施细则 国家助学贷款管理实施细则 学生校内勤工助学管理办法 临时困难补助发放管理规定 家庭经济困难学生减免学费和住宿费暂行办法 生活补贴发放规定 饮水 洗澡 电话补助发放办法 毕业生学费和国家助学贷款代偿实施细则 应征入伍服义务兵役国家资助实施细则 退役士兵学费资助实施细则  
整体思路是 通过奖学金 助学金 贷款 勤工助学 减免学费和临时补助等多条渠道 保障家庭经济困难学生可以顺利完成学业

---

# 条目: 国家奖学金与国家励志奖学金定位
# 类别: 制度
# 主题: 国家奖学金
# 来源: 首都师范大学学生手册整理

面向本专科学生的国家级最高层次奖励项目 标准名称为 国家奖学金 主要奖励在思想品德和学业成绩等方面都特别优秀的全日制在校生 一般要求学习成绩名列前茅 综合测评优良 且无违法违纪记录 具体条件由教育部文件和学校国家奖学金实施细则共同规定  
面向品学兼优且家庭经济困难学生的国家级奖励项目 标准名称为 国家励志奖学金 主要面向学习成绩较好 自立自强 家庭经济困难的全日制本专科生 既强调德育和学习表现 也强调经济困难情况 具体标准由国家政策和学校实施细则执行  
两类奖学金每学年评定一次 通过个人申请 班级民主评议 院系审核 学校评审和公示等环节产生 获奖学生名单会在校内一定范围公示

---

# 条目: 国家助学金与北京市国家助学金
# 类别: 制度
# 主题: 国家助学金
# 来源: 首都师范大学学生手册整理

面向家庭经济困难学生的生活补助项目 标准名称为 国家助学金 主要用于补贴学生在校期间的基本生活开支  
首都师范大学执行国家助学金和北京市国家助学金两类制度 具体实施细则包括资助对象范围 资助标准 评定条件 评审程序和发放方式等  
一般要求学生具有我校正式学籍 按时注册 家庭经济困难 通过学校认定 且遵守校纪校规 积极上进  
资助标准和名额根据当年国家和北京市文件以及学校下达的指标执行

---

# 条目: 社会奖助学金基本规则
# 类别: 制度
# 主题: 社会奖助学金
# 来源: 首都师范大学学生手册整理

由社会机构 校友 企业等出资设立的奖学金或助学金 统称为社会奖助学金 例如金宝奖学金 尚朴奖学金等 其评选条件原则上分别参照学校二等及以上优秀或三等奖学金的条件执行  
申请社会奖助学金的学生 既要满足出资单位提出的具体要求 也要满足学校奖助学金的基本评选条件  
这类奖助学金通常每学年评定一次 一般在十月中下旬进行 由学生本人提出申请 班级资助评议小组进行民主评议 多数同意后方可推荐  
院系资助工作领导小组对班级推荐名单进行初审 对符合条件的学生进行三日公示 无异议后报学生处学生资助管理中心  
学校学生资助管理中心根据院系初审结果综合评定 由学校学生资助工作领导小组研究通过后 在校内公示五日 公示无异议后报相关出资单位 一般在当年十二月底前发放到学生个人

---

# 条目: 临时困难补助与学费住宿费减免概况
# 类别: 制度
# 主题: 困难补助与减免
# 来源: 首都师范大学学生手册整理

临时困难补助是一种应对家庭突发重大变故 意外事故 或短期经济困难的专项资助 由学生资助管理中心负责具体实施  
临时困难补助通常由学生本人提出申请 经过院系初审后报学生处审批 申请金额在三千元以下的 由院系报学生处审批 三千元以上的 由院系报学生处 再经学生处和财务处核实后 报主管校领导审批 通过后由学生本人或院系老师到财务处领取  
另外 学费和住宿费减免由学校出资 专门针对家庭经济特别困难 但仍坚持完成学业的全日制本专科生 有单独的暂行办法 文号为学发〔2021〕7号 规定了减免对象 条件 和申请程序  
整体目标是通过一次性补助与长期减免相结合 减轻家庭经济特别困难学生的学费住宿费压力

---

# 条目: 校内勤工助学与生活补贴
# 类别: 制度
# 主题: 勤工助学与补贴
# 来源: 首都师范大学学生手册整理

学校通过校内勤工助学岗位和多种生活补贴支持家庭经济困难学生  
勤工助学管理办法对岗位设置 招聘条件 工作时间 报酬标准和安全管理等做出规定 一般要求不影响正常学习 工作时间和强度有上限 以帮助学生合理安排学习与工作  
生活补贴相关规定明确了补贴对象 发放标准和发放方式 包括普通生活补贴 和饮水 洗澡 电话等专项补助  
这类补贴与国家助学金 学费减免 临时困难补助一起 构成经济资助体系的重要补充 部分补贴会与家庭经济困难认定结果挂钩

---

# 条目: 公费医疗报销基本规则
# 类别: 制度
# 主题: 公费医疗
# 来源: 首都师范大学学生手册整理

符合条件的学生可以按学校公费医疗管理办法报销部分医疗费用  
在门诊进行大型检查时 如计算机断层扫描 磁共振 彩超 超声心动图 冠脉造影等 单项检查费用在二百元及以上的 一般需要事先由校医院业务副院长审批 急诊和住院情况可以例外 未按规定事先审批的 检查费用中有部分需要个人自负  
在检查和治疗中 单项费用超过五百元的一次性医用材料被视为贵重医用材料 对此类材料费用 按规定比例报销 其余部分个人承担  
使用贵重医用设备或材料时 需由经治医生填写贵重医用设备及材料检查与治疗审批表 由本人或家属签字 并经公费医疗管理办公室审批备案 后续产生的费用才能按规定比例报销 特殊治疗如伽玛刀治疗 有单独的报销比例要求

---

# 条目: 校心理素质教育与咨询资源总览
# 类别: 制度
# 主题: 心理健康资源
# 来源: 首都师范大学学生手册整理

学校心理素质教育与咨询中心已建设多年 目前面积约七百五十平方米 拥有个体咨询室 团体辅导室 音乐治疗室 电话咨询室 沙盘治疗室 图书资料室等多种功能用房 并配置了生物反馈仪 咨询督导监控设备等专业仪器  
中心现有专职教师若干人 兼职教师和咨询师若干人 包括教授 副教授 医院心理科医师 以及取得国家或行业认证的注册督导师 注册心理师 国家二级心理咨询师等  
中心承担心理健康教育课程 心理讲座和团体辅导等工作 为全校师生提供心理咨询 危机干预和心理健康宣传  
总体定位是为师生提供专业 安全的心理支持环境

# 条目: 心理健康教育五级网络结构
# 类别: 制度
# 主题: 心理危机干预网络
# 来源: 首都师范大学学生手册整理

学校建立了多层级的心理健康教育和危机干预支持系统 一般被概括为五级网络  
第一层是学校层面 由学校主管领导 心理健康教育与危机干预领导小组 学生处 研究生工作部 保卫处 校团委 校医院以及心理素质教育与咨询中心等组成 负责制定相关文件 了解全校学生心理状况 统筹心理教育和危机干预工作  
第二层是院系层面 由院系领导 学工干部和心理骨干组成 负责贯彻学校文件 组织院系心理教育活动 选拔和培训学生心理委员 了解和上报本院系学生心理问题  
第三层是班级层面 班主任 任课教师和班级心理骨干承担日常观察 宣传和初步干预职责 班主任要掌握本班学生心理状况 熟悉常见心理问题 接受心理培训 并及时与心理中心和院系沟通  
后续层级覆盖宿舍 朋辈和家庭 联系学校心理中心和院系 一起参与学生心理危机干预  
整体目标是 发现问题要早 报告渠道要通畅 干预措施要有组织

---

# 条目: 校内外心理求助渠道与联系方式
# 类别: 制度
# 主题: 心理求助渠道
# 来源: 首都师范大学学生手册整理

学校在手册中列出了多种心理咨询和心理援助渠道  
校内方面 良乡校区心理中心位于艺术楼一层 预约电话为心理中心公布的专线 工作日白天提供咨询和危机干预 校本部心理中心位于大学生活动中心一层西侧 有专门的预约电话 白天和晚间均有咨询时段 危机个案可以在工作时间内随时处理  
校外方面 包括北京青少年服务台 电话为一二三五五 提供心理和法律咨询 有明确的工作时间 还包括北京心理危机干预热线 免费电话和面向手机用户的号码 七天二十四小时提供自杀危机干预和抑郁情绪疏导 服务对象既包括当事人也包括亲友  
此外 学校还列出合作医院的心理或精神科门诊联系方式 便于需要药物或住院治疗的同学就医  
这些信息适合在生存助手中作为紧急求助和长期咨询的推荐渠道
