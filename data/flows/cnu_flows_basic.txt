# 条目: 新生保留入学资格办理流程
# 类别: 流程
# 主题: 新生保留入学资格
# 来源: 首都师范大学学生手册整理

第一步 新生本人确认需要保留入学资格的原因  
常见原因包括因病经指定医院诊断不宜在校学习 或应征入伍等  
原因需要符合学校规定的情形  

第二步 提出书面申请  
新生向所在学院提交书面申请 说明原因 保留时间等内容  
同时填写学校统一使用的新生入学资格变动审批表  

第三步 准备并提交证明材料  
因病保留需要附上学校指定医院出具的诊断材料  
因应征入伍等原因保留的 按学校要求提供相关证明  

第四步 学院审核并签署意见  
学院主管领导审核材料 核实情况 在审批表上签署意见  

第五步 学校审批  
招生就业等职能部门汇总材料 按规定权限批准保留  
批准后学校为学生办理保留入学资格手续  

第六步 期满申请入学  
保留入学资格期满前 新生需在规定期限内向学校申请入学 因病保留的还需提供康复证明 经指定医院诊断和校医院复查合格后 才能正式入学

# 条目: 学生休学申请与审批流程
# 类别: 流程
# 主题: 休学
# 来源: 首都师范大学学生手册整理

第一步 学生确认休学原因  
常见原因包括因伤病治疗时间较长 或因其他正当事由一学期请假时间超过六周  
原因应当符合学校规定的休学条件  

第二步 提交休学申请  
学生向所在学院提交书面休学申请 说明原因 拟休学起止学期等内容  
因病休学需附上学校指定医院出具的诊断证明  

第三步 学院审核  
学院对申请和材料进行审核 必要时联系医院或家长核实情况  
审核通过后在休学申请材料上签署意见  

第四步 教务部门备案与批准  
学院将审核材料报送教务部门备案 按学校流程完成休学批准手续  
休学一般以一年为期 累计时间和次数不得超过学校规定上限  

第五步 办理离校手续  
完成休学审批后 学生按学校要求办理退宿等相关手续 离校休养或处理个人事宜  
休学期间保留学籍 但不享受在校学生待遇  

---

# 条目: 学生复学申请流程
# 类别: 流程
# 主题: 复学
# 来源: 首都师范大学学生手册整理

第一步 在休学期满前做好复学准备  
学生应注意学校规定的复学申请时间 一般在复学前一学期末或规定时间内提出申请  

第二步 提交复学申请  
学生向所在学院提交书面复学申请 说明休学期间情况及复学意愿  

第三步 提交相关证明材料  
因病休学的学生 需提供学校指定医院出具的诊断证明 证明已具备在校学习生活条件  
因其他原因休学的学生 需提供与当初原因相对应的情况说明或证明材料  

第四步 学院审核  
学院审核申请和材料 对疑点进行核实 并提出是否同意复学意见  

第五步 校医院复查与教务审批  
因病休学的学生 还需通过校医院复查 确认符合体检要求  
学院审核意见和校医院结论一并报教务部门审批  

第六步 编入相应年级  
复学获批后 根据学生已有课程和学分情况 将其编入原专业相应年级  
学生按新年级安排继续学习

# 条目: 本科生转专业办理流程
# 类别: 流程
# 主题: 转专业
# 来源: 首都师范大学学生手册整理

第一步 学生确定转专业意向  
学生结合自身兴趣 专长和学业情况 确定拟转入专业  
需要注意 是否在学校规定的可申请时间段内 一般为第一学年及第二学年的第二学期  

第二步 向原学院提出申请  
学生向所在学院提交书面转专业申请 说明理由 学习情况及今后规划  

第三步 原学院审核  
原学院对申请人的学习成绩 学习态度等进行审核  
确认学生对现专业确有不适 或对目标专业有稳定兴趣和一定基础  

第四步 拟转入学院考核  
学生按拟转入学院要求 参加测试或面谈等环节  
拟转入学院根据容量和培养要求 决定是否同意接收  

第五步 学校层面公示与批准  
拟转入名单经学校相关部门审核后 在校内一定范围公示  
公示无异议后 由学校按权限批准办理转专业  

第六步 办理学籍与课程转换  
获得批准后 学生按规定时间到相关部门办理学籍信息变更手续  
对原专业已修且与新专业课程相同的 学校认定后承认学分 学生后续按新专业培养方案修读课程  

---

# 条目: 学生转学办理流程
# 类别: 流程
# 主题: 转学
# 来源: 首都师范大学学生手册整理

第一步 学生本人提出转学申请  
学生根据个人情况 确认存在疾病 特殊困难或确实无法适应原学校培养要求等正当理由  
向拟转入学校提交书面申请 说明转学原因 与拟转专业等  

第二步 原学校审核同意转出  
拟从首都师范大学转出的学生 需向所在学院和学校提出申请  
学校审核理由 是否符合国家和学校关于不得转学情形的规定  
同意转出的学校出具相关证明材料  

第三步 拟转入学校审核条件  
拟转入学校负责审核申请人的学习情况 高考成绩 是否达到本校相应专业录取标准 以及培养条件等  
认为符合条件且学校有培养能力的 提交校长办公会或专题会议研究决定  

第四步 办理教育行政部门备案  
跨省转学的 由转出地省级教育行政部门与转入地省级教育行政部门协调 按条件确认后办理转学手续  
需要变更户口的 由有关部门将文件抄送转入学校所在地公安机关办理户籍迁移  

第五步 完成学籍关系变更  
转学批准后 学生在规定时间内到原学校办理离校手续 并到转入学校办理报到和学籍接收手续  
转入学校自转学完成后三个月内向所在地教育行政部门备案

# 条目: 学业试读申请与解除流程
# 类别: 流程
# 主题: 学业试读
# 来源: 首都师范大学学生手册整理

第一步 学院或教务部门发出学业预警  
当学生首次出现按学期计算平均学分不足十二学分时 学校依据规定向学生发出学业预警  
学院告知学生需要办理试读手续  

第二步 学生提出试读申请  
学生填写试读申请材料 说明个人原因和改进计划  
由辅导员或班主任签署意见 后报学院主管领导审核  

第三步 学院审批并报教务部门备案  
学院根据学生情况决定是否同意试读 同意的报教务部门备案  
试读期一般为一年  

第四步 试读期间跟踪与指导  
试读期间 学院加强对学生学习情况的跟踪和指导  
学生应按计划修读课程 尽量提高平均学分  

第五步 期满审核是否解除试读  
试读期满时 将期间取得学分与既往学分合并 重新按学期平均学分计算  
达到十二学分及以上的 可以按规定程序申请解除试读状态  
未达到标准 或再次出现平均学分不足十二学分的 属于应予退学的情形  

---

# 条目: 学生退学离校办理流程
# 类别: 流程
# 主题: 退学离校
# 来源: 首都师范大学学生手册整理

第一步 明确退学原因和依据  
退学情形包括学业未达要求 学习年限届满未完成学业 休学期满不复学 等多种情况  
可以是学校按规定作出的退学处理 也可以是学生本人提出退学申请  

第二步 退学决定与告知  
对于学校作出退学处理的 由学院提出意见 教务部门审核 校长办公会研究决定  
学校出具书面退学决定书 并送达学生本人  

第三步 学生办理离校手续  
收到退学决定或本人申请获批后 学生需在规定时间内办理退宿和财务结算等离校手续  
逾期未办理的 学校可按有关规定处理  

第四步 档案和户口处理  
学校按照规定将退学学生档案退回其家庭所在地  
户口按国家相关规定迁回原户籍地或家庭户籍所在地  

第五步 相关证明材料  
在校学习达到一定时间的退学学生 学校向其出具退学证明和成绩证明  
在校学习满一年及以上的学生 还可以获得肄业证书  

---

# 条目: 学生申诉办理流程
# 类别: 流程
# 主题: 学生申诉
# 来源: 首都师范大学学生手册整理

第一步 学生查阅处理或处分决定  
学生收到学校作出的处理或处分决定书后 如有异议 需要在有效期限内仔细阅读决定内容 确认事实 依据和处分种类  

第二步 准备书面申诉材料  
在接到决定书之日起十日内 学生可以向学校学生申诉处理委员会提出书面申诉  
申诉书中应写明基本情况 申诉请求 事实与理由 并附相关证据材料  

第三步 学生申诉处理委员会受理  
申诉处理委员会在受理后 对相关事实和依据进行复查 必要时听取学生和相关部门的说明  
一般在十五日内做出复查结论 情况复杂的 经批准可以适当延长 并制作复查决定书  

第四步 告知复查结果  
复查决定书送达申诉学生 载明复查意见和进一步申诉期限  
如认为原处理存在事实 依据或程序不当 可以建议撤销或变更 交由相关职能部门按程序重新提交校长办公会研究  

第五步 向教育行政部门申诉  
学生对学校复查决定仍有异议的 可以在接到复查决定书之日起十五日内 向北京市教育委员会提出书面申诉  
教育行政部门按规定受理并在一定期限内作出处理决定  

第六步 申诉期满视为放弃  
自处理 处分或复查决定书送达之日起 学生在申诉期内未提出申诉的 视为放弃申诉权利 后续原则上不再受理同一事项的校内申诉
