# 条目: 学制与最长学习年限总览
# 类别: 制度
# 主题: 学制与学习年限
# 来源: 首都师范大学学生手册

首都师范大学本科生学制为四年 最长学习年限为六年 包括休学和保留学籍时间  
专升本学生学制为三年 最长学习年限为五年  
专科生学制为二年 最长学习年限为三年  
因应征参加中国人民解放军的 本科生 专升本学生 专科生的最长学习年限可以延长 对应为八年 七年 六年  
因创业休学的 本科生 专升本学生 专科生的最长学习年限可以适当延长 对应为七年 六年 四年  
港澳台侨学生的最长学习年限为七年  

---

# 条目: 新生入学报到与保留入学资格
# 类别: 制度
# 主题: 新生入学
# 来源: 首都师范大学学生手册

按国家招生规定录取的新生 必须在学校规定的期限内持录取通知书到校办理入学手续  
因正当原因不能按期报到的 需要事先向学校请假 未请假或者请假逾期达到两周 且没有不可抗力等正当理由的 视为放弃入学资格  
学校在新生报到时会先进行入学资格初步审查 审查合格的予以注册学籍 如发现录取材料与本人情况不符 或存在违反招生规定的情形 学校有权取消入学资格  
新生因疾病 经学校指定医院诊断并由校医院复查证明不宜在校学习 或因应征入伍等原因 可以申请保留入学资格 保留期限一般为一年 应征入伍的可保留至退役后两年 保留入学资格期间不具有学籍  

---

# 条目: 新生入学复查与取消学籍情形
# 类别: 制度
# 主题: 新生复查
# 来源: 首都师范大学学生手册

学生入学后三个月内 学校按照国家招生规定对新生进行复查 主要包括录取手续是否合规 录取资格是否真实 本人与档案信息是否一致 身心健康状况是否符合要求 等内容  
复查不合格者 学校根据情况给予相应处理 直至取消入学资格  
凡属弄虚作假 徇私舞弊取得学籍的 无论时间先后 一经查实 立即取消学籍 情节严重的 移交有关部门处理  
如复查中发现学生身心状况不适宜在校学习 经学校指定医院诊断 需要在家休养的 可以按保留入学资格的规定执行  

---

# 条目: 学期注册与暂缓注册
# 类别: 制度
# 主题: 注册
# 来源: 首都师范大学学生手册

每学期开学时 学生必须按学校规定办理注册手续  
因故不能按时注册的 需要依照规定办理暂缓注册手续  
未按规定缴纳学费或不符合注册条件的 学校可以不予注册  
家庭经济困难学生可以申请国家助学贷款或其他形式资助 办理相关手续后可以完成注册  
学校按照国家规定建立学生资助体系 保证学生不因家庭经济困难而放弃学业  

---

# 条目: 考勤与缺课后果
# 类别: 制度
# 主题: 考勤
# 来源: 首都师范大学学生手册

教学计划规定的各类教学活动包括课堂教学 实验 实习 毕业论文或毕业设计 社会实践 军事训练 政治学习等 都需要进行考勤  
任课教师应当采取适当方式记录考勤 情况计入平时成绩  
学生缺课时间达到本课程总教学时间三分之一及以上的 不得参加该课程期末考核  
学生不能按时参加教学活动的 必须事先请假并获得批准 未获批准缺席的 按旷课处理  

---

# 条目: 成绩构成与及格标准
# 类别: 制度
# 主题: 成绩与学分
# 来源: 首都师范大学学生手册

课程考核分为考试和考查两类 必修课一般采用考试 选修课可采用考试或考查  
学校采用百分制或五级分制评定成绩  
百分制中 六十分及以上为及格 取得相应学分  
五级分制中 及格及以上为通过 取得相应学分  
许多课程的总评成绩由平时成绩和期末成绩共同组成 平时成绩一般含作业 单元测试 到课情况等 通常占百分之五十至百分之六十 期末成绩占百分之四十至百分之五十  
未选课或未按规定参加课程学习的 不得参加该课程考核 擅自参加考核的 成绩和学分不予承认  

---

# 条目: 重修与免修基本规则
# 类别: 制度
# 主题: 重修与免修
# 来源: 首都师范大学学生手册

在校期间 必修课程总评成绩不及格的 学生必须在下一年级选课并跟班重修  
重修时学生应参加课程全部教学环节和考核 重修课程成绩按实际分数记载 并在成绩单中标注为重修 同时注明重修次数  
因病或其他不可抗力原因未能参加考核导致不及格的 也需要通过重修重新取得学分  
因健康原因不能参加体育等课程学习的学生 经校医院和相关部门审批 可以办理免修 课程学分予以承认 成绩记为免修  

---

# 条目: 学分绩点与平均学分绩点
# 类别: 制度
# 主题: 绩点
# 来源: 首都师范大学学生手册

学校采用平均学分绩点反映学生学习情况  
平均学分绩点可以按学期 学年或整个修业年限计算 一般以学生在某时间段内所修全部课程成绩为基础  
课程成绩记为无效的 在计算绩点时按零分处理 免修课程不计入绩点计算  
百分制成绩与绩点之间有对应关系 成绩越高 绩点越高 五级分制等级与百分制之间也有对应区间  

---

# 条目: 转专业条件与时间
# 类别: 制度
# 主题: 转专业
# 来源: 首都师范大学学生手册

本科生在学习期间对其他专业有稳定兴趣和一定专长的 可以申请转专业  
学生因入学后发现有某些疾病 经学校指定医院证明不适宜在原专业继续学习 但能够在本校其他专业学习的 可以申请转入其他专业  
学校根据社会需求和专业调整情况 允许在读学生在相关专业之间适当流动  
休学创业或退役后复学的学生 因个人情况确需转专业的 学校在审核时予以优先考虑  
学生一般在第一学年及第二学年的第二学期可以提出转专业申请 超出时间范围的原则上不再受理 转专业后执行转入专业的培养方案 原专业已修且与新专业课程相同的 经学校认定后承认学分  

---

# 条目: 转学条件与限制
# 类别: 制度
# 主题: 转学
# 来源: 首都师范大学学生手册

学生一般应在被录取学校完成学业 因患病或特殊困难等客观原因 无法在原学校继续学习 或确实不适应原学校培养要求的 可按规定申请转学  
申请转学需由学生本人提出书面申请 说明理由 经原学校和拟转入学校审核同意 后由转入学校按程序提交校长办公会研究决定  
有几类情形不得转学 包括 入学未满一学期或毕业前一年 高考成绩低于拟转入学校相关专业同一生源地录取成绩 由低学历层次转为高学历层次 以定向就业形式录取以及缺乏正当转学理由等情况  

---

# 条目: 休学条件与休学年限
# 类别: 制度
# 主题: 休学
# 来源: 首都师范大学学生手册

学生因伤病需要集中治疗 一学期治疗时间超过六周 或因其他正当事由 一学期请假时间累计超过六周的 应当办理休学  
休学一般以一年为一周期 本科生休学时间累计不超过二年 且休学次数不超过两次 专科生休学时间累计不超过一年 且休学次数不超过一次 创业休学学生累计休学时间不超过三年 次数不超过三次  
休学时间计入学校规定的最长学习年限 但不计入在校学习时间  
休学期间学生保留学籍 但不享受在校学生待遇 因伤病休学的医疗费用按照学校相关规定处理  

---

# 条目: 复学基本要求
# 类别: 制度
# 主题: 复学
# 来源: 首都师范大学学生手册

休学期满的学生 需要在规定时间内向学校提出复学申请  
因病休学的 复学时应提供学校指定医院出具的诊断证明 并经校医院复查合格 教务部门审批后方可复学 如发现诊断证明存在伪造 或复查不合格 不予复学  
因其他原因休学的 复学申请需附相关证明材料 由所在学院审核后报教务部门批准 复学后根据已修课程情况编入原专业相应年级学习  
休学期间有严重违法违纪行为的 学校可以拒绝其复学 休学期满未在规定期限内办理复学手续的 视为放弃学籍  

---

# 条目: 学业试读与平均学分不足
# 类别: 制度
# 主题: 学业预警
# 来源: 首都师范大学学生手册

学生在校学习期间 第一次出现按学期计算的平均学分少于十二学分时 需要办理学业试读 试读期为一年  
试读期间取得的学分与之前各学期学分合并 重新按学期计算平均学分 若达到十二学分及以上 可解除试读状态  
因转专业或转学转入下一年级学习的学生 在计算平均学分时 在校学习学期数可以相应减少 两个学期 休学学期不计入在校学习学期数  
首次平均学分不足十二学分又未办理试读 或试读期满仍未达到解除标准 再次出现平均学分不足十二学分的 均属于应当退学的情形  

---

# 条目: 退学情形与后续处理
# 类别: 制度
# 主题: 退学
# 来源: 首都师范大学学生手册

学生有以下情形之一 学校应按规定办理退学  
第一次出现平均学分不足十二学分但未申请试读  
试读期满仍未达到解除试读标准  
试读解除后再次出现平均学分不足十二学分  
在规定学习年限内未完成学业  
休学或保留学籍期满 未在规定期限内提出复学申请 或复查不合格  
经指定医院诊断 因疾病或意外伤残不能继续在校学习  
未经批准连续两周未参加学校规定教学活动  
超过两周未注册且未办理暂缓注册手续  
以及学校规定的其他应予退学的情形  
退学学生需要在规定期限内办理退宿和离校手续 档案退回家庭所在地 户口按国家有关规定迁回原户籍地或家庭户籍所在地 在校学习满一年及以上的学生可获得肄业证书  

---

# 条目: 毕业 结业与肄业的基本区分
# 类别: 制度
# 主题: 毕业与结业
# 来源: 首都师范大学学生手册

学生在规定学习年限内 修完培养方案规定的全部内容 成绩合格 达到学校毕业要求 经学院审查和教务部门复查合格后 准予毕业 并颁发毕业证书 符合学位授予条件的 颁发学士学位证书  
在规定学习年限内修完规定内容但未达到毕业要求的 准予结业 发给结业证书 未达到结业要求的 按退学处理 发给肄业证书  
结业学生离校后 在一定年限内 可以通过重修 实习补做 或毕业论文补作等方式达到毕业要求 后续可以申请颁发毕业证书和学位证书  

---

# 条目: 学业证书管理与撤销条件
# 类别: 制度
# 主题: 学位与证书管理
# 来源: 首都师范大学学生手册

学校按照招生时确定的办学类型和学习形式 以及学生录取时填写的个人信息 填写和颁发学历证书 学位证书及其他学业证书  
学生在校期间变更姓名或出生日期等证书信息时 需要提供有法定效力的证明 由学校审核  
对违反招生规定取得入学资格或学籍的 学校取消其学籍 不颁发学历证书和学位证书 已发证书的 学校依法撤销并注销注册信息  
对以作弊 剽窃 抄袭等学术不端行为或其他不正当手段获得学历证书 学位证书的 学校依法撤销证书 被撤销证书自公告起视为无效  
学历证书和学位证书遗失或损坏的 学生可申请学校出具相应证明书 证明书与原证书具有同等效力  

---

# 条目: 学生纪律处分的种类与原则
# 类别: 制度
# 主题: 处分种类
# 来源: 首都师范大学学生手册

学校对违反法律法规和校纪校规的学生 以教育为主 教育与惩戒相结合  
纪律处分分为五种 警告 严重警告 记过 留校察看 开除学籍  
处分决定需要依据事实和证据 种类和幅度要与行为性质和后果严重程度相适应  
在对学生作出处分或其他不利决定前 学校应当告知事实 理由和依据 并告知学生享有陈述和申辩权利  
除开除学籍外 一般处分期限为六个月到十二个月 期满按程序办理解除处分 解除后学生在表彰 奖励和其他权益方面不再受原处分影响  

---

# 条目: 旷课对应的处分标准
# 类别: 制度
# 主题: 旷课处分
# 来源: 首都师范大学学生手册

学生一学期内旷课累计达到二十学时 学校给予警告处分  
累计达到三十学时 给予严重警告处分  
累计达到五十学时 给予记过处分  
累计达到六十学时 给予留校察看处分  
经常旷课 累计达到八十学时 经教育不改的 给予开除学籍处分  
学院应当及时对旷课学生进行教育和提醒 对情节严重的学生依照上述标准处理  

---

# 条目: 开除学籍的典型情形
# 类别: 制度
# 主题: 开除学籍
# 来源: 首都师范大学学生手册

学生有以下情形之一 学校可以给予开除学籍处分  
违反宪法 反对基本政治原则 破坏安定团结 扰乱社会秩序  
触犯国家法律 构成刑事犯罪  
受到治安管理处罚 情节严重 性质恶劣  
代替他人或让他人代替自己参加考试 组织作弊 使用通讯设备等器材作弊 向他人出售试题或答案牟利 或有其他严重考试作弊行为  
在学位论文或公开发表的研究成果中存在严重抄袭 篡改 伪造或买卖论文等学术不端行为  
严重破坏学校教育教学秩序 生活秩序或公共场所管理秩序  
侵害他人或组织合法权益造成严重后果  
屡次违反学校规定 受处分后经教育仍不改正  

---

# 条目: 学生申诉与外部救济渠道
# 类别: 制度
# 主题: 学生申诉
# 来源: 首都师范大学学生手册

学校设立学生申诉处理委员会 负责受理学生对各类处理或处分决定不服的申诉  
学生对学校处理或处分决定有异议的 自接到决定书之日起十日内 可以向申诉处理委员会提交书面申诉  
申诉处理委员会一般在十五日内完成复查并作出复查决定 情况复杂的 经批准可适当延长 并可提出暂缓执行原决定的建议  
学生对复查决定仍有异议的 在接到复查决定书之日起十五日内 可以向北京市教育委员会提出书面申诉  
处理 处分或复查决定书送达后 学生在申诉期内未提出申诉的 视为放弃申诉  
学生认为学校或工作人员违反规定侵害其合法权益 或校内规章与上位法冲突的 可以向教育行政部门投诉
